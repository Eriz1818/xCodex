# @eriz1818/xcodex-responses-api-proxy

<p align="center"><code>npm i -g @eriz1818/xcodex-responses-api-proxy</code> to install <code>xcodex-responses-api-proxy</code></p>

This package distributes the prebuilt [xCodex Responses API proxy binary](https://github.com/Eriz1818/xCodex/tree/main/codex-rs/responses-api-proxy) for macOS, Linux, and Windows.

To see available options, run:

```
node ./bin/xcodex-responses-api-proxy.js --help
```

Refer to [`codex-rs/responses-api-proxy/README.md`](https://github.com/Eriz1818/xCodex/blob/main/codex-rs/responses-api-proxy/README.md) for detailed documentation.
